# Paradigm Desktop Releases

## Current team preview

[Download Paradigm 0.1.0 for Apple Silicon Macs](https://github.com/Paradigm-Study/paradigm-desktop-releases/releases/download/preview-0.1.0-4902bb2-20260713/Paradigm-0.1.0-UNSIGNED-PREVIEW-arm64.dmg)

> **Warning:** This build is an unsigned, unnotarized internal preview for Paradigm teammates who trust its source. It requires an Apple Silicon Mac running macOS 14 or newer, and macOS will require a manual security override before first launch. Automatic updates are disabled in this build.

Download the DMG, drag Paradigm into Applications, try opening it once, then use **System Settings → Privacy & Security → Open Anyway**. See [Apple's instructions](https://support.apple.com/guide/mac-help/open-an-app-by-overriding-security-settings-mh40617/mac) and the [preview release page](https://github.com/Paradigm-Study/paradigm-desktop-releases/releases/tag/preview-0.1.0-4902bb2-20260713) for checksums and verification details.

Signed and notarized consumer releases will appear here after Apple Developer enrollment is available.
